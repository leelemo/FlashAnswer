# FlashAnswer — 从零到手机安装完整指南

## 前提条件

| 需要 | 说明 |
|------|------|
| GitHub 账号 | 免费注册 https://github.com |
| Apple ID | 普通苹果账号即可，不需要付费开发者账号 |
| AltStore | Windows 端工具，用于签名安装 .ipa |
| iPhone | iOS 17+ |

---

## 第一步：安装 Git（Windows）

1. 下载 https://git-scm.com/download/win
2. 一路默认安装
3. 打开「Git Bash」验证：`git --version`

---

## 第二步：创建 GitHub 仓库

1. 登录 https://github.com → 右上角 **+** → **New repository**
2. 名称填 `FlashAnswer`，选 **Private**（私有）
3. 点 **Create repository**
4. 复制仓库地址（形如 `https://github.com/你的用户名/FlashAnswer.git`）

---

## 第三步：推送代码到 GitHub

打开「Git Bash」，执行以下命令（把路径替换成实际路径）：

```bash
cd "D:/NasHP同步syDrive/workbuddy/2026-06-21-09-42-21/FlashAnswer"
git init
git add .
git commit -m "Initial commit: FlashAnswer"
git branch -M main
git remote add origin https://github.com/leelemo/FlashAnswer.git
git push -u origin main
```

输入 GitHub 用户名和密码（或 Personal Access Token）后完成推送。

---

## 第四步：等待 GitHub Actions 编译

1. 打开仓库页面 → 点击顶部 **Actions** 标签
2. 看到「Build FlashAnswer iOS」正在运行（黄色圆圈）
3. 等待约 **15~25 分钟**（macOS runner 排队+编译时间）
4. 绿色勾 = 编译成功

---

## 第五步：下载 .ipa 文件

1. 点进刚刚完成的 Actions 运行记录
2. 页面底部找到 **Artifacts** 区域
3. 点击 **FlashAnswer-ipa** 下载 zip 文件
4. 解压得到 `FlashAnswer.ipa`

---

## 第六步：安装 AltStore（Windows）

1. 下载 AltServer for Windows：https://altstore.io/
2. 安装 iTunes（Microsoft Store 版不行，用 Apple 官网版）：
   https://www.apple.com/itunes/download/win64
3. 安装 iCloud（Apple 官网版）：
   https://updates.cdn-apple.com/2020/windows/001-39935-20200911-1A70AA56-F448-11EA-8CC0-99D41950005E/iCloudSetup.exe
4. 安装 AltServer，系统托盘会出现一个菱形图标

---

## 第七步：在 iPhone 安装 AltStore

1. iPhone 用 USB 连电脑，**信任此电脑**
2. 点击系统托盘的 AltServer 图标 → **Install AltStore** → 选你的 iPhone
3. 输入 Apple ID 和密码（只在本地使用，不会上传）
4. iPhone 上打开 **设置 → 通用 → VPN与设备管理** → 信任 AltStore

---

## 第八步：用 AltStore 安装 FlashAnswer

1. iPhone 打开 AltStore → 底部 **My Apps** → 左上角 **+**
2. 选择刚才下载的 `FlashAnswer.ipa` 文件
3. 输入 Apple ID，等待安装完成
4. **设置 → 通用 → VPN与设备管理** → 信任 FlashAnswer

---

## 续签（每 7 天一次）

AltStore 可以**自动续签**：
- iPhone 和电脑在同一 WiFi
- 后台运行 AltServer（系统托盘）
- AltStore 会在证书到期前自动刷新

也可以手动：AltStore → My Apps → FlashAnswer → **Refresh**

## 使用说明

### 准备 Excel 题库

格式要求（.xlsx）：

| A列（题目） | B列（选项） | C列（答案） |
|-----------|------------|-----------|
| 中国首都是哪里？ | A.上海 B.北京 C.广州 | B |
| 1+1=? | | 2 |

- 第一行为表头（自动跳过）
- B列选项可以为空
- 支持导入多个题库（会自动去重合并）

### 使用流程

1. 打开 FlashAnswer
2. 点「**导入题库**」→ 选择 .xlsx 文件
3. 点「**开始监听**」
4. 将手机放在桌上，麦克风朝向题目声音
5. 识别到题目后，通知栏自动弹出答案
6. 划掉通知或等待 20 秒，自动开始下一轮
7. 可随时点「立即重听」手动重置

## 常见问题

**Q：第一次打开提示权限？**  
A：点允许，需要麦克风权限和通知权限才能正常工作。

**Q：匹配不准？**  
A：题库中题目描述要和实际朗读的内容接近。可以把题目导入多种表述方式。

**Q：后台被杀了？**  
A：App 使用静音音频保活，正常情况下不会被杀。如果被杀，重新打开点「开始监听」即可。

**Q：编译失败？**  
A：在 GitHub Actions 页面点进失败的运行，查看错误日志，截图发给我。
