# FlashAnswer

语音答题助手 — iOS App

## 功能

- 导入 Excel 题库（.xlsx，A列题目/B列选项/C列答案）
- 后台持续语音监听，实时匹配题目
- 通知栏推送答案
- 本地倒排索引 + 编辑距离模糊匹配

## 要求

- iOS 17+
- iPhone（建议 iPhone 12 及以上，支持设备端语音识别）

## 编译

直接 push 到 `main` 分支，GitHub Actions 自动编译。详见 `操作手册.md`。

## 安装

使用 AltStore（Windows）侧载安装，无需 Apple Developer 账号。详见 `操作手册.md`。
